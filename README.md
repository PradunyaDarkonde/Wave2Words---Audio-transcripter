# 🎧 Wave2Words – Audio Transcripter

A full-stack web application that allows users to upload audio files and receive accurate transcriptions using **OpenAI's Whisper API**. The backend is built with **Java Spring Boot**, while the frontend is developed using **HTML**, **CSS**, and **JavaScript**, providing a smooth and responsive user experience.

## 📌 Project Overview

Wave2Words simplifies speech-to-text conversion by offering a clean UI for uploading audio files. The backend processes these files and communicates with OpenAI’s Whisper API to return high-quality transcriptions in real time. It’s ideal for anyone needing fast and reliable voice-to-text capabilities.

## 🧰 Technologies & Tools Used

### 🛠️ Backend
- **Java 17** – Core backend language.
- **Spring Boot** – For building and managing the REST API.
- **Spring Web** – To handle HTTP requests and responses.
- **Multipart File Handling** – For processing user-uploaded audio files.
- **CORS Configuration** – Enables frontend-backend integration across origins.
- **JUnit 5 / Spring Boot Test** – For testing backend components.
- **Maven** – Build automation and dependency management.

### 🎨 Frontend
- **HTML5** – UI structure and layout.
- **CSS3** – Styling and responsive design.
- **JavaScript (Vanilla)** – Handles interactivity, file uploads, and API calls.
- **Fetch API** – To communicate with the backend asynchronously.

### 🤖 External Services
- **OpenAI Whisper API** – For accurate speech-to-text transcription.

## 🚀 Features

- 🎙 Upload audio files (MP3, WAV, M4A)
- ⚙️ Real-time transcription using OpenAI Whisper
- 🧠 Handles large files efficiently
- 🖥 Clean and user-friendly frontend interface
- 🌐 Fully responsive design
- 🧪 Error handling and backend testing setup
